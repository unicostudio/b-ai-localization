Getting Started

Installation

Clone the repository and install dependencies:

git clone https://github.com/unicostudio/b-ai-localization.git
cd b-ai-localization
pip install -r requirements.txt

Running the Application

python app.py

Then open your browser and navigate to http://127.0.0.1:5001

CSV Input Format

Your CSV file should include these essential columns:

IDS;EN;LOCID
ID1;Tap on the biggest flower.;LEVEL_TEXT_1
ID1;Drag out the Sun behind Lily's head.;HINT_1_1
ID2;Lets find Tricky Lily;LEVEL_TEXT_2

Environment Setup

Create a .env file with your OpenRouter API key:

OPENROUTER_API_KEY=your_api_key_here

How It Works

  1. Upload your CSV file with game text
  2. Select the directory containing game screenshots
  3. Choose target languages and AI model
  4. The tool extracts text from images using OCR
  5. Vision AI generates descriptions of game screenshots
  6. LLM creates context-aware translations
  7. Download the generated files when processing completes
